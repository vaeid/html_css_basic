توضیحات درباره تگ <img>

تگ <img> برای نمایش تصاویر در صفحات وب استفاده می‌شود. این تگ به صورت خودبسته است و نیازی به تگ بسته ندارد. برخی از ویژگی‌های مهم این تگ عبارتند از:

  • src: آدرس تصویر (URL) که می‌خواهید نمایش دهید.
  • alt: متن جایگزین برای تصویر. این متن در صورتی که تصویر بارگذاری نشود یا برای کاربران با ناتوانی‌های بصری نمایش داده می‌شود.
  • width و height: برای تعیین ابعاد تصویر (اختیاری).
  • class: برای اضافه کردن استایل‌های CSS به تصویر (اختیاری).

حالت‌های دیگر تگ <img>

می‌توانید از ویژگی‌های دیگر تگ <img> نیز استفاده کنید:

  • استفاده از style برای اضافه کردن استایل‌های inline.
  • استفاده از title برای نمایش توضیحات اضافی هنگام قرار دادن ماوس بر روی تصویر.
  • استفاده از loading برای کنترل بارگذاری تصویر (مثل lazy برای بارگذاری تنبل).
